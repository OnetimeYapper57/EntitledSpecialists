# EntitledSpecialists
A RimWorld mod for 1.6 that removes Ideology's role work restrictions, but increases the base expectations of affected roles.<br/>
Requires: Ideology.<br/>
Some optional features use the following mods:<br/>
Vanilla Memes Expanded - Memes and Structures<br/>
Alpha Memes<br/>
Mort's Ideologies Empiricism and Faith<br/>
